# tucano-strategies

Coleção de estratégias de exemplo / referência para o ecossistema **Tucano**.

Inclui (features opcionais):
- momentum
- mean_rev (mean reversion)
- microstructure
- options

Uso:
```toml
[dependencies]
tucano-strategies = { version = "0.1", features = ["momentum"] }
```
